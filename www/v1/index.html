FlexVue (플랙스뷰)

FlexVue 는 프론트앤드 개발용 API 입니다.
css 는 TailWind Css를 사용하며
javascript 는 바닐라 스크립트(es6 +) 만을 이용해 만들어 졌습니다.

웹 (Web) / 하이브리드 앱(Application)을 위해 만들어 졌습니다.

반응형 홈페이지와 하이브리드앱 제작을 위한
최대 60fps 의 에니메이션과 함께 어플리케이션 만들기에 최적화 되어 있습니다.

비동기 AsyncTask

서버 데이터와 템플릿을 사용하기 위해 비동기(async & wait)를 사용 합니다. 

                                        new AsyncTask().doGet( '접속경로', '전송데이터' ).then(resp => {
    
                });

다이나믹 import

하이브리드앱을 위한 다이나믹 import 를 사용할 수 있도록 지원합니다. 

                                    new AsyncTask().doImport( include_file ).then(Module => {

            });

리터럴 템플릿 (Template)

리터럴 자바스크립트 HTML 템플릿을 지원합니다. 

            <script type="text/template" id="tpl_example">
            </script>

다이나믹 import CSS

CSS 파일을 다이나믹 import 할 수 있습니다. 

                                    new AsyncTask().doImportCss( include_css ).then(()=>{
                                        Log.d('completed');
                                    });
